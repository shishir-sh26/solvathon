# PLACEMENT PRO

Placement Pro is an AI-powered recruitment ecosystem designed for 24-hour hackathons. It streamlines the connection between Training & Placement Officers (TPOs), students, and alumni.

## 🚀 Features
- **TPO Dashboard:** Smart drive management and automated student filtering.
- **Student Portal:** ATS-optimized resume builder and application tracking.
- **Jarvis Bot:** An AI agent that orchestrates platform data via natural language.
- **Alumni Connect:** Peer mentorship and referral hub.
- **Virtual Interview:** Real-time sentiment and confidence analysis.
- **Market Intelligence:** Insights into skill gaps and market trends.

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Zustand, Lucide React.
- **Backend:** FastAPI (Python), Pydantic, Supabase (PostgreSQL).
- **AI/ML:** Google Gemini API, Langchain, PyMuPDF.

## 🛠️ Setup Instructions

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. Configure `.env` with your Keys.
4. `uvicorn app.main:app --reload`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
